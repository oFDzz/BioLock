BioLock — App Lock with Face ID / Passcode + Stealth Mode
==========================================================
Last updated: 2026-05-16
Device: iPhone 14 Pro (A16, arm64e), iOS 15.6.1, Dopamine (rootless)
Repository: https://github.com/oFDzz/BioLock

=====================================
WHAT IT DOES
=====================================
1. Locks selected apps behind Face ID authentication (home screen, app
   switcher, and notification taps)
2. Blurs locked app content in the app switcher
3. Stealth Mode: rapid sleep button presses hide selected apps from the
   home screen entirely — they vanish until you toggle stealth off

=====================================
HOW IT WORKS
=====================================
All hooks run in SpringBoard (com.apple.springboard).

App Lock:
  1. Hooks SBIconView._handleTap → intercepts icon taps
  2. Hooks SBMainSwitcherViewController app selection → intercepts switcher
  3. Hooks SBFluidSwitcherItemContainer.didMoveToWindow → adds blur overlay
  4. Hooks notification dispatch → intercepts notification taps
  5. If target app is locked → Face ID (biometrics + passcode fallback)
  6. Auth success → original action proceeds
  7. Auth fail/cancel → blocked, stay where you are

Stealth Mode:
  1. Hooks SpringBoard lock/power button handler
  2. Tracks rapid presses within a time window (configurable 3-6 clicks)
  3. On trigger → toggles stealth state
  4. Hooks SBLeafIcon/SBApplicationIcon visibility → hides stealth apps
  5. Reloads icon model → apps disappear/reappear from home screen
  6. Double haptic feedback on activation, single on deactivation
  7. Blurs stealth-hidden apps in switcher too

Preferences Gate:
  - BioLock settings panel requires Face ID before showing content
  - Blur overlay covers UI until authenticated
  - Failed auth → popped back to Settings

=====================================
PREFERENCE PANEL DESIGN
=====================================
Section 0 — Master Toggle
  Enable/disable BioLock with switch

Section 1 — Protected Apps
  Shows only currently locked apps
  Swipe left to remove protection ("Unlock")
  Long press for context menu (add to stealth)

Section 2 — Stealth Mode
  Enable toggle + click count picker (3-6 rapid presses)

Section 3 — Stealth Hidden Apps
  Collapsible list of apps hidden during stealth
  Swipe left to unhide

Section 4 — App Library
  Collapsible "Add Apps..." button
  Search bar filters the list
  Tap to lock, long press for context menu:
    - Lock with Face ID
    - Hide in Stealth Mode
    - Lock + Hide in Stealth (both)

=====================================
FILES
=====================================
Tweak.x (~250 lines)
  - Hook 1: SBIconView._handleTap (icon tap)
  - Hook 2: SBMainSwitcherViewController (switcher selection)
  - Hook 3: SBFluidSwitcherItemContainer (switcher blur)
  - Hook 4: NCNotificationDispatcher (notification tap)
  - Hook 5: SpringBoard lock button (stealth trigger)
  - Hook 6: SBLeafIcon visibility (stealth hide)
  - Shared auth function with Face ID + passcode fallback

BioLockPrefs/BLRootListController.m
  - PSViewController subclass with redesigned UI
  - 5 sections: toggle, protected, stealth, stealth apps, app library
  - Collapsible app library (no more infinite scroll)
  - Swipe-to-delete for quick removal
  - Context menus for multi-action (lock/hide/both)
  - Face ID gate on entry

BioLock.plist — Filter: com.apple.springboard only

=====================================
PREFERENCES
=====================================
Stored in: /var/jb/Library/Application Support/BioLock/prefs.plist
  enabled          — master toggle (BOOL)
  lockedApps       — array of bundle ID strings (Face ID locked)
  stealthEnabled   — stealth mode toggle (BOOL)
  stealthHiddenApps — array of bundle ID strings (hidden in stealth)
  stealthClickCount — number of rapid clicks to trigger (3-6, default 4)
